# 문제 해결 / 수동 확인 (TROUBLESHOOTING)

이 문서는 두 부분으로 나뉩니다.

- **A. 수동 확인 필요 항목** — 티스토리에서 직접 테스트해야만 검증되는 구조 항목 (아직 미반영, P1)
- **B. 증상별 해결** — 적용 중 흔히 마주치는 문제

---

## A. 수동 확인 필요 항목 (manual Tistory preview check)

아래 항목은 오프라인에서 확정할 수 없어 **현재 스킨에 반영하지 않았습니다.** 티스토리에서 직접 테스트한 뒤 별도로 보강할 예정입니다.

| 항목 | 현재 상태 | 확인 방법 |
|---|---|---|
| **검색(search)** | 헤더 검색폼이 `action="/search"` + `name="q"` (`/search?q=...`) 방식 | 실제 검색 시 결과가 나오는지 확인. 티스토리가 `/search/{키워드}` 경로를 요구하면 폼 수정 필요 |
| **방명록(guestbook)** | 전용 섹션 없음 | 방명록 페이지를 열어 정상 렌더 여부 확인. 비어 보이면 전용 블록 추가 검토 |
| **빈 상태(empty state)** | "글 없음/결과 없음" 안내 없음 | 빈 카테고리·검색 결과 0건 페이지에서 본문 영역이 비는지 확인 |
| **태그(tags)** | 글 상세에 태그 표시 없음 | 글에 태그가 필요한지 결정 후 보강 |
| **이전/다음 글** | `[##_article_prev_link_##]` / `[##_article_next_link_##]` 사용 (skin.html 주석 ⚠) | 실제 글에서 링크가 채워지는지 확인. 비면 `[##_article_rep_prev_link_##]` 계열로 교체 검토 |
| **사이드바 모듈** | `[##_sidebar_element_##]` 사용 (skin.html 주석 ⚠) | 사이드바에 모듈이 나오는지 확인. 안 나오면 `[##_sidebar_##]` 로 교체 검토 |
| **광고(ad) 배치** | 전용 광고 슬롯/수익 치환자 없음 | 광고 설정 시 어디에 삽입되는지, 다크 배경과 충돌하지 않는지 확인 |
| **보호된 글(비밀글)** | 별도 처리 없음 | 비밀번호 입력 폼이 본문 영역에 정상 표시되는지 확인 |
| **미리보기 이미지** | 파일 미포함, `index.xml` 은 `./images/preview.jpg` 참조 | 이미지를 직접 추가 (INSTALL.md 3번) |

> ⚠️ 위 항목을 수정할 때는 **치환자/조건 블록 구조를 함부로 바꾸지 마세요.** 잘못 건드리면 글 전체가 렌더링되지 않을 수 있습니다.

## B. 증상별 해결

### 1. 글 목록/본문이 아예 안 보인다
- **원인:** 글 모듈 구조 손상. 글 목록·본문은 반드시 `<s_article_rep>` 안에 `<s_index_article_rep>` 와 `<s_permalink_article_rep>` 가 들어 있어야 합니다.
- **해결:** `skin.html` 구조가 아래와 같은지 확인하세요.
  ```html
  <s_article_rep>
    <s_index_article_rep> ... </s_index_article_rep>
    <s_permalink_article_rep> ... </s_permalink_article_rep>
  </s_article_rep>
  <s_paging_rep> ... </s_paging_rep>   <!-- 모듈 밖, 형제 위치 -->
  ```

### 2. 스킨 저장 시 "필수 치환자 누락" 경고
- **원인:** `<s_t3>` 컨테이너나 필수 치환자 누락.
- **해결:** `<body id="[##_body_id_##]">` 바로 뒤에 `<s_t3>` 가 열리고 `</body>` 직전에 `</s_t3>` 가 닫히는지 확인.

### 3. 별 배경이 안 보인다
- **원인:** `images/script.js` 가 로드되지 않음(경로 문제).
- **해결:** 파일이 `images/script.js` 로 업로드됐는지, `skin.html` 끝의 `<script src="./images/script.js">` 경로와 일치하는지 확인.

### 4. 커스텀 커서가 안 나온다
- **원인:** `images/space-cursor.cur` 미업로드 또는 경로 불일치.
- **해결:** 파일을 `images/` 에 업로드. (없어도 기본 커서로 폴백되므로 치명적이지 않음)

### 5. 사이드바에 마스코트만 있고 메뉴가 없다
- **원인:** 티스토리 사이드바 모듈 미설정.
- **해결:** 관리자 → 꾸미기 → 사이드바에서 카테고리/검색/최근 글 모듈 추가. (INSTALL.md 4번)

### 6. 마스코트 선택이 저장되지 않는다
- **원인:** 브라우저 시크릿/프라이버시 모드에서 `localStorage` 차단.
- **해결:** 코드는 차단 시에도 오류 없이 기본 마스코트(`🧑‍🚀`)로 동작합니다. 저장 유지가 필요하면 일반 모드에서 확인.

### 7. 마스코트가 사람+로켓 2글자로 쪼개져 보인다
- **원인:** OS/브라우저 폰트가 우주인 ZWJ 이모지(`🧑‍🚀`)를 단일 글리프로 지원하지 않음.
- **해결:** OS 이모지 폰트 문제로, 스킨 버그가 아닙니다. 최신 OS에서는 정상 표시됩니다.

### 8. 한글 라벨이 너무 작다 / 픽셀 폰트로 안 보인다
- **원인:** Press Start 2P 에는 한글 글리프가 없어 모노스페이스로 폴백됩니다.
- **해결:** 의도된 동작입니다. 크기는 `.section-label` / `.comments-title` / `.sb-station` / `.post-nav-label` 에서 조정. (CUSTOMIZE.md 3번)

### 9. 목록 구분선("탐사 기록")이 글 상세에서도 보이거나 안 보인다
- **원인:** `.section-header` 는 `.main:has(.mission-card)` (CSS `:has()`) 로 목록 페이지에서만 표시됩니다.
- **해결:** 구형 브라우저는 `:has()` 미지원으로 라벨이 안 보일 수 있습니다(장식 요소라 기능 영향 없음). 최신 브라우저에서 확인.

### 10. 미리보기 썸네일이 비어 있다
- **원인:** `images/preview.jpg` 미추가.
- **해결:** 미리보기 이미지를 직접 추가. (INSTALL.md 3번) 블로그 동작에는 영향 없음.

---

문서로 해결되지 않는 구조 문제는, 수정 전에 반드시 백업 후 **치환자/조건 블록을 보존**한 채 최소 변경으로 접근하세요.
